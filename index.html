Major Projects

My major projects include a CRUD application called Yelp-Camp that I created with a full stack development setup. I utilized nodejs, javascript, html, css, and bootstrap. I used the NoSQL database framework called Mongoose and set up a local and cloud-based database connection using cloudinary for images and Mongo cloud storage. The features of the project include a login and verification system, authentication when users add or remove campground reviews for campgrounds they have visited, and a review system for reviewing each individual campground. There are many more nuanced features as well that I would be happy to explain if you have any questions regarding the project.

I worked on a personalized project in a software engineering class to learn and utlilize the agile methodology for software development. Me and my team of 5 other developers focused heavily on the scrum process and we decided to pursue a sizeable video game project. The game is inspired by traditional rogue-like games that utilize ASCII graphics and we used swing and an open-source github project called AsciiPanel to help us build it. I contributed to every part of this project aside from the inventory system. My largest contributions consisted of the procudural generation algorithm for the randomized maps that are featured in the game and also assistance with the collision based combat system and handling the integration with each feature into the main game.

I have created many more projects for my course work including many of the common data structures and algorithms that are used as well as several personal projects. In my portfolio section I have included the projects I mentioned linked to my github and you may find many other projects I have worked on there as well.
